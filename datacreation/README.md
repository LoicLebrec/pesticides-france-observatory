# Pesticides France Observatory
Cartographie nationale des pesticides en France (2023) avec visualisation Plotly et table interactive.

## Description
Ce dépôt contient un script Python qui génère une carte interactive et un tableau des volumes de pesticides par commune en France pour l'année 2023. Les données sont issues de fichiers CSV publics et de ressources GeoJSON.

## Utilisation
1. Placez les fichiers CSV requis dans le dossier `data/` (voir le script pour les noms attendus).
2. Exécutez le script principal :
   ```bash
   python 2023dta.py
   ```
3. Le fichier HTML sera généré à la racine du projet.
